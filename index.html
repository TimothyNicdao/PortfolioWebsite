CommonLit Competition

Evaluated the readability of texts using transformers

CommonLit Readability Competition

For this project, the task was to evaluate the readability of text and rate them based on their difficulty. The target values that I tried to predict were the Bradley-Terry analysis of pairwise comparisons between excerpts. Teachers spanning grades 3-12 served as the raters for these comparisons with majority being teachers for grades 6-10.

In order to evaluate the readability of the texts, I used a hugging face Transformer Roberta model to extract features of the text and added a regression head for the down stream task. The model was implemented with PyTorch and to determine model generalizability, I used a Kfold cross validation technique with 5 folds.

ICS 432: Multithreaded Image Processing App

A multithreaded image processing app for the class ICS 432: Concurrency

ICS 432: Concurrency Project

This project was built for the class ICS 432: Concurrency and was implemented in Java. Our task was to build an image processing app that uses concurrency to speed up processing of batches of images. For this project, we had to implement several methods that allow for concurrency without allowing race conditions by using Locks, Condition Variables and Semaphores.

AniGAN

Creating synthetic anime images using GANs.

AniGAN

For this project, I created synthetic anime images using Generative Adversarial Networks, specifically using the Deep Convolutional GAN (DCGAN) architecture. The image dataset consisted of anime images from Danbooru of size 512x512 pixels.
